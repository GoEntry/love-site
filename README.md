# ❤️ Love Landing (Vue)

Небольшой интерактивный лендинг, созданный на **Vue 3**, как романтический подарок.
Сайт содержит мини-историю, воспоминания, письмо и несколько интерактивных элементов.

Проект развёрнут на **GitHub Pages**, поэтому его можно открыть по ссылке и изучить.

---

# ✨ Возможности

Сайт включает несколько интерактивных элементов:

* ❤️ падающие сердечки на фоне
* 💥 взрыв сердечек при клике по экрану
* 🎵 встроенный музыкальный плеер
* ✨ плавные переходы между страницами
* 💌 печатающееся письмо
* ⏱ таймер отношений (сколько мы уже вместе)
* 🔒 секретная кнопка с модальным окном
* 🌫 blur фона при открытии модального окна
* 📱 адаптивная верстка для телефонов

---

# 📄 Страницы сайта

* **Начало** — приветственный экран
* **Наша история** — короткая история отношений
* **Фото** — страница с воспоминаниями
* **Мы вместе** — живой таймер отношений
* **Письмо** — романтическое письмо с анимацией печати
* **Любовь** — финальная страница с секретной кнопкой

# 🚀 Запуск проекта

Установка зависимостей:

```bash
npm install
```

Запуск dev-сервера:

```bash
npm run dev
```

Сборка проекта:

```bash
npm run build
```

# 💖 Идея проекта

Этот проект был создан как небольшой интерактивный подарок.
